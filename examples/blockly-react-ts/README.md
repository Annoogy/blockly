# blockly-react-ts-sample [![Built on Blockly](https://tinyurl.com/built-on-blockly)](https://github.com/google/blockly)

This sample shows how to load Blockly in a [React](https://reactjs.org/) project.

## Running the sample

### Installation

```
npm install
```

### Running

```
npm run start
```

### Browse

Open [http://localhost:5173/](http://localhost:5173/)
